教室人脸签到系统（项目介绍）

基于摄像头与人脸识别技术，实现教室出勤自动化管理的最小可行产品（MVP）。支持人脸检测、特征提取、识别比对、活体检测、签到去重与报表导出。

MVP 语言 / 技术：Python · FastAPI · OpenCV · face_recognition / insightface 部署：Edge（Raspberry/Jetson）或服务器（Docker）

项目概述

本项目旨在为中小学校或培训机构提供一套轻量、可落地的教室人脸签到解决方案。通过摄像头实时识别在场学生并自动生成签到记录，减轻教师手工点名负担，同时支持签到报表、缺勤统计与基础防作弊能力。

核心功能

  • 人脸检测与对齐（静态图片 & 视频帧）
  • 人脸特征提取与比对（Embedding）
  • 签到记录：按节次/时间窗去重、CSV 导出
  • 活体检测（基础动作检测/眨眼/张嘴）
  • 管理端：学生注册（Enrollment）、签到查询
  • 部署支持：Docker 一键启动 + 可选边缘部署方案

系统架构（简要）

  • 摄像头采集层（Edge）：RTSP/USB 摄像头，负责采集与本地预处理
  • 识别服务（可部署于边缘或云端）：人脸检测 → 对齐 → 提取 embedding → 比对
  • 应用后端：FastAPI 提供 enroll/recognize/attendance API、存储（SQLite/Postgres）
  • 前端展示：简单的管理界面与实时识别页面（HTML/JS 或 React）

技术选型（建议）

Python 3.9+
OpenCV
insightface / face_recognition
FastAPI + Uvicorn
SQLite / PostgreSQL
Docker

隐私与合规

  • 优先存储人脸 embedding，尽量不长期保存原始照片。
  • 数据最小化与保留策略：建议默认保留 1 年，超期删除或匿名化。
  • 接口与数据访问采用 HTTPS 与认证（建议 JWT）。
  • 在部署前须取得学校/家长的书面同意并遵守当地数据保护法规。

运行示例（快速开始）

开发环境准备后，示例命令（项目根目录）：

  • python -m venv .venv，source .venv/bin/activate
  • pip install -r requirements.txt
  • 启动开发服务器：uvicorn app:app --reload --host 0.0.0.0 --port 8000
  • 用浏览器打开：http://localhost:8000 （或前端页面）

常见 API：

  • POST /api/enroll — 注册学生（student_id + images）
  • POST /api/recognize — 上传帧并返回识别结果
  • GET /api/attendance — 查询签到记录

项目亮点

  • 零基础可复现的教学路线与代码示例
  • 支持边缘部署，降低带宽与隐私风险
  • 模块化设计，易替换检测/识别模型
  • 包含活体检测与签到去重策略
